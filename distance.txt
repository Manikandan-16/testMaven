1)Write a Java program find the distance between 2 words.(i.e number of words between those 2 words in the given sentence).
Scenario to Validate
i) Sentence and finding words will get as input from program
ii) Validate the both words should be provided If those words are not present in the sentence then print as "Invalid word" 
iii) Print the finding words and distance if it is present on sentence.




import java.util.*;

import java.lang.*;

import java.io.*;

class Distance{
	
public static void main (String[] args) throws java.lang.Exception
	
{

Scanner sc=new Scanner(System.in);

int dist=0;

String s=sc.nextLine();

String str1 = sc.nextLine();
String str2 = sc.nextLine();

if(s.indexOf(str1)==-1||s.indexOf(str2)==-1)

System.out.println("Invalid Word");
else

{

dist=Math.abs(s.indexOf(str1)-s.indexOf(str2));

System.out.println(str1);
System.out.println(str2);
System.out.println(dist-1);

}
}
}
